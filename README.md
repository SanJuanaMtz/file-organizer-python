# File Organizer – Python Project Evolution

This repository contains the development of my Python file organizer through multiple versions.

## Version 1

Basic file organization logic.

## Version 2

Improved structure and file classification.

## Version 3 – File Organizer Pro

Desktop application built with Tkinter.

Features:

- Select folder
- Organize files by extension
- Automatically create folders
- Display movement logs

## Technologies

- Python
- Tkinter
- os
- shutil

## Goal of the Project

This project was created while learning programming concepts through CS50 and focuses on automation, file handling, and improving code structure through iteration.
